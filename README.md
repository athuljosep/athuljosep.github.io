## Projects
- **Energy Resiliency of a building using Reinforcement Learning** Project employs a Zap Q-learning RL controller to ensure maximum and critical loads are served during grid blackouts caused by extreme weather, using 7-day weather and load forecasts to optimize decisions based on the availability of PV and Tesla Powerwall resources.
- **EnergyPlus GUI App Development** Plotly Dash-based web application has been developed as a GUI tool for accessing EnergyPlus, enabling generation, aggregation, and visualization using PNNL prototype data to simulate various scenarios based on user input.
- **Distance Protection Schemes under High Renewable Penetration** A distance protection technique is proposed for transmission lines in high-renewable scenarios, with DSP implementation and Hardware-in-the-Loop testing.
- **Coordinated Control Strategy for Frequency Support in Wind Turbine Generators** Project proposes a control strategy to enable inertial response from Doubly Fed Induction Generators in coordination with Super-conducting Magnetic Energy Storage Systems with minimized secondary effects.
- **Simulation and Prototyping of Dual Rotor Induction Motor** A co-axial dual squirrel cage rotor induction motor is discussed with potential application in electric vehicles. A conceptual analysis and mechanical configuration of this motor is presented in this project.
- **Light Sensitive Emergency Lamp** Energy saving is given high priority in this project. This is done by making sure that the emergency turns on only when absolutely required. Automatic adaption to natural light conditions is added using an LDR.
- **Smart Switch** Innovation of conventional switch into reliable touch switch. Any appliance can be controlled by a single touch.
